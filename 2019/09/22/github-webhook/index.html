添加WebHooks

登录github，进入到你的博客仓库。点击Settings -> Webhooks->Add web hook

你要输入三个信息：Payload URL（你接收的webhook钩子请求的接口）、Content type（数据传输方式）、Secret（校验信息）

然后下面你可以选择在什么行为下，激活这个webhook钩子发请求给你的服务器。你可以选择第一项，只在推送代码到远程仓库时，通知你的服务器拉取最新代码。
